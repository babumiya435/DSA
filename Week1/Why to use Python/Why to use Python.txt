trade off between javascript, python

Javascript:
    - arr[i]
    - arr.push(1) // pushing element to end
    - arr.pop() // poping element from end
    - arr.unshift(100) // pushing elements to starting
    - arr.shift() // poping starting element
    - arr.slice / arr.splice(index, how many elements to remove, insert new element in the pace of removal)

Python:
    - arr[i]
    - arr.append(2)
    - arr.pop() # pop the last index element
    - arr.insert(i, value) # insert new element at index
    - remove - pop(i)
