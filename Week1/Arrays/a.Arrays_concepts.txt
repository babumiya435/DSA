Common terms: Subarray(elements with ordered sequesnce), Subsequence(elements without order)

Time Complexity:
access using index - O(1)
search using value - O(n)
search(sorted array) - O(log n) - binary search
insert - O(n)
insert at the end - O(1) - array.push(value)
insert at the start - O(n) - array.unshift(value)
remove - O(n)
remove from the end - O(1) - array.pop()
remove from the start - O(n) - array.shift()
